# pokedex

A simple application with the list of pokemons.

## Start

### Установите rearguard

```
npm i -g rearguard
```

### Склонируй репозиторий

```
git clone git@github.com:mmastya/food-controler.git
```

### Установите модули

```
npm i
```

### Выполните

```
npm run boostrap
```

### Запустите

```
npm start
```

### Собрать проект

```
npm run build -- --release
```
